# SSPI Sponsorship Documents

This repository hosts SSPI's sponsorship and media kit pages via GitHub Pages.

## Files

- `sspi-digital-sponsorship-2026.html` — Digital and Online Sponsorship Opportunities 2026
- `sspi-media-kit-2026.html` — Better Satellite World Media Kit 2026

## Images Required

Upload these files into an `images/` folder in this repository:

| Filename | Description |
|---|---|
| `sspi-white-logo.png` | SSPI logo, white letters, transparent background (SSPI_cmyk_white_letters.png) |
| `ssf-logo.png` | Space and Satellite Futures circular badge (SSF_Transparent_Logo.png) |

Both files are referenced as `images/filename.png` in the HTML.
